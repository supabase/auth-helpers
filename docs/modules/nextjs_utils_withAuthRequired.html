  • default(options?: { getServerSideProps?: GetServerSideProps<{}, ParsedUrlQuery, PreviewData>; redirectTo?: string }): (context: GetServerSidePropsContext<ParsedUrlQuery, PreviewData>) => Promise<any>
    • Defined in src/nextjs/utils/withAuthRequired.ts:45

    If you wrap your getServerSideProps with withAuthRequired your props object will be augmented with the user object {@link User}

    // pages/profile.js
    import { withAuthRequired } from '@supabase/supabase-auth-helpers/nextjs';

    export default function Profile({ user }) {
      return <div>Hello {user.name}</div>;
    }

    export const getServerSideProps = withAuthRequired({ redirectTo: '/login' });

    If there is no authenticated user, they will be redirect to your home page, unless you specify the redirectTo option.

    You can pass in your own getServerSideProps method, the props returned from this will be merged with the user props. You can also access the user session data by calling getUser inside of this method, eg:

    // pages/protected-page.js
    import { withAuthRequired, getUser } from '@supabase/supabase-auth-helpers/nextjs';

    export default function ProtectedPage({ user, customProp }) {
      return <div>Protected content</div>;
    }

    export const getServerSideProps = withAuthRequired({
      redirectTo: '/foo',
      async getServerSideProps(ctx) {
        // access the user object
        const user = await getUser(ctx);
        return { props: { email: user!.email } };
      }
    });
